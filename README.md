# Send-a-bot

Copy the project on the catkin workspace
```
$ cd catkin_ws
$ cd src
$ git clone
```

Install depenancies
``` 
$ rosdep update
$ cd ..
$ rosdep update rosdep install --rosdistro noetic --ignore-src --from-paths src
$ catkin_make
```

```
roslaunch pioneer3at demo_navigation_gmapping.launch
```
